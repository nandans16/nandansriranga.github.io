Selected Projects

Sequential Universal Two-Sample Testing with Adaptive and Active Sensing

Developed sequential universal tests for comparing two unknown data-generating distributions without assuming parametric likelihood models. Designed adaptive weighting and active stream-selection policies with theoretical error guarantees. This work connects to online model evaluation, distribution-shift detection, adaptive data collection, and monitoring ML systems under changing data distributions.

Online Sensor Collaboration for Moving Target Detection

Developed adaptive sensor-collaboration algorithms for moving-target detection in distributed sensor networks. Built MATLAB simulation pipelines for Monte Carlo experiments, parameter sweeps, detection probability, false-alarm analysis, and topology tradeoffs. Used MATLAB Parallel Computing Toolbox and parfor loops to accelerate large-scale simulations.

Quickest Detection of Link Outages in Wireless Sensor Networks

Developed low-delay statistical detectors for identifying abrupt link failures and sparse heterogeneous anomalies in wireless sensor networks. The work is relevant to time-series anomaly detection, predictive maintenance, sensor-network monitoring, and cyber-physical system health diagnostics.

Passive Radar Target Detection with Inter-Receiver Collaboration

Developed distributed passive radar detection methods using whitening, subspace transformations, cross-correlation, and receiver collaboration. Evaluated detection performance under communication constraints and weak uplink conditions.

AI Agent for Progressive Chess

Built an AI agent for progressive chess using Monte Carlo Tree Search with UCT-based exploration, a custom chess-position heuristic, and a convolutional neural network for state-value priors. This project reflects my interest in building computational systems from first principles and improving them through experimentation and debugging.
